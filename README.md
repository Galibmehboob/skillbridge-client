# 🚀 SkillBridge

SkillBridge is a modern full-stack collaboration platform where founders, developers, designers, and freelancers can connect, showcase their skills, and collaborate on real projects.

## 🌐 Live Demo

- **Frontend:** https://your-vercel-url.vercel.app
- **Backend:** https://your-render-url.onrender.com

---

## ✨ Features

- 🔐 Secure Authentication with Better Auth
- 👤 User Registration & Login
- 📝 User Profile Management
- 💼 Add, Update & Manage Skills
- 🔍 Explore Developers & Skills
- 🤝 Collaboration Request System
- ✅ Accept / Reject Collaboration Requests
- 📱 Fully Responsive Design
- 🌙 Modern UI with Tailwind CSS & HeroUI
- 🛡️ Admin Dashboard
- 👥 User Management
- 🚫 Block & Delete Users
- 📊 Dashboard Statistics

## ⚙️ Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Client

```bash
cd skillbridge-client

npm install

npm run dev
```

### Server

```bash
cd skillbridge-server

npm install

npm run dev
```
